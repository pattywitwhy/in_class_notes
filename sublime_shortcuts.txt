sublime_shortcuts.txt

cmd + ctrl + <up/down arrow>  # moves the line up or down in your code
cmd + shift + p               # Searches something? I don't get it
cmd + l (no highlight)        # Highlights the whole line
cmd + c (no highlight)        # Copies the whole line
cmd + option 1                # Show 1 window only
cmd + option + shift 2        # Show 2 windows, side-by-side
cmd + option + shift 2        # Show 2 windows, top-and-bottom
cmd + option + 3              # 3 windows side-by-side
cmd + option + 5              # 4 windows, 1 in each corner 
cmd + ]                       # Shift your code to the right
cmd + []                      # Shift your code to the left
cmd + click                   # adds multiple cursors

When defining a class, w then tab -> adds a writer
When defining a class, r then tab -> adds a reader
